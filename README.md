# Studio UI

A modernist pure CSS framework.

Studio UI is still in an early draft phase.

## Documentation

All Studio classes are prefixed with `s-`.

line-height + 2 * padding will be referred to as *item height*

### Core

#### s-vv
vertical view

#### s-vh
horizontal view

#### s-b
button

#### s-li
list item

### Quick Layout

#### s-[width]
for [100, 200, 300, 400, 500]
width = width

#### s-lh
height = *item height*

#### s-wh
wrap children width

#### s-wv
wrap children height

#### s-al
left align text

#### s-ac
center align text

#### s-ar
right align text

#### s-aj
justify text

#### s-ns
no select
